Welcome to edgeNet

edgeNet is a distributed edge cloud, in the family of PlanetLab, GENI, Canada’s SAVI infrastructure, Japan’s JGN-X, Germany’s G-Lab, and PlanetLab Europe. It is a modern distributed edge cloud, incorporating advances in Cloud technologies over the past few years.

edgeNet is based on industry-standard Cloud software, with Docker as the containerization technology and Kubernetes as the node manager and deployment solution. It is an opt-inglobal Kubernetes cluster; once a user has authenticated with this portal and been approved, she will be able to use standard Kubernetes tools and technologies to deploy an application across the edgeNet infrastructure
